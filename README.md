# Tiny Markdown Editor

## Introduction

Tiny Markdown Editor is a simple desktop Markdown editor.

The current goal of this project is to provide a lightweight and straightforward Markdown editing experience, with a small codebase that can be extended step by step.

## Tech Stack

- Rust
- Tauri
- Vue 3
- TypeScript
- Vite
- vue-i18n

## Run

Development:

```sh
./run.sh
```

Windows:

```bat
run.bat
```

## Release Build

Linux:

```sh
./publish.sh
```

Windows:

```bat
publish.bat
```
